# AI Engineer Workshop: From Approval Loops to Autonomous Agents with Docker

A 90-minute hands-on workshop on Docker primitives for safe AI agent autonomy. Covers SBX (Docker Sandboxes), DHI (Docker Hardened Images), MCP, SBX Kits, Docker Agent multi-agent orchestration, and per-role tool isolation.

Co-presented by Dan Ndombe and John Craft at the AI Engineer's World's Fair in San Francisco, June 29 to July 2, 2026.
